# Coffee Shop E-commerce Website

Welcome to the Coffee Shop E-commerce project! This is a modern, responsive web application built with React, Vite, and Tailwind CSS.

## Features

- **Premium UI Design**: A rich, coffee-themed aesthetic.
- **Responsive Layout**: Works seamlessly on mobile and desktop.
- **Product Menu**: Browse a variety of coffee drinks and pastries.
- **Shopping Cart**: Add items and manage your order.
- **Animations**: Smooth transitions powered by Framer Motion.

## Tech Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS (v4)
- **Icons**: Lucide React
- **Animations**: Motion (framer-motion)

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
